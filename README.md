# Sistema de Gestión de Restaurante

## Autor

Sam G

## Descripción

Este proyecto fue desarrollado en Python utilizando Programación Orientada a Objetos.

El sistema permite gestionar productos y clientes de un restaurante mediante una estructura modular.

## Estructura del proyecto

```
restaurant_app/
│
├── modelos/
│   ├── producto.py
│   └── cliente.py
│
├── servicios/
│   └── restaurante.py
│
├── main.py
└── README.md
```

## Tipos de datos utilizados

- str
- int
- float
- bool
- list

## Clases

### Producto

Representa un producto disponible en el restaurante.

### Cliente

Representa un cliente registrado.

### Restaurante

Administra las listas de productos y clientes.

## Reflexión

Este proyecto permitió aplicar identificadores descriptivos, tipos de datos básicos, listas y una estructura modular. Además, reforzó el uso de clases, objetos y buenas prácticas de programación en Python.
